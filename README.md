a shitty repo for shitty use

## How to use it ?
In **`tokens.txt`** :
```txt
<token>:voice channel id>:<status>:<muted>:<deafen>:<camera>:<stream>
```
> Status can be : online, dnd, idle, invisible
> Muted, deafen and camera is "yes" or "no"
> Sream, "yes" or "no"

## Requirement:

- NodeJS +v18
